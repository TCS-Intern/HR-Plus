- Buttons: Primary, Secondary, Ghost, Destructive
- Cards with hover states
- Form inputs with validation states
- Status badges (draft, active, pending, complete)
- Progress indicators
- Data tables with sorting/filtering
- Modal dialogs
- Toast notifications

---

## User Personas

### Hiring Manager (Primary)
- Creates job requisitions
- Reviews shortlisted candidates
- Makes final hiring decisions
- Approves offers

### Recruiter
- Manages candidate pipeline
- Uploads CVs
- Schedules assessments
- Coordinates with candidates

### Candidate (External)
- Completes video assessments
- Receives/responds to offers

## Mobile Responsiveness

### Breakpoints
```
Mobile:    < 640px   (single column, stacked)
Tablet:    640-1024px (collapsible sidebar)
Desktop:   > 1024px  (full layout)
```

### Mobile-specific Considerations
- Sidebar becomes bottom navigation
- Tables become card lists
- Video recorder full-screen
- Voice input prominent

---

## Accessibility Requirements

- WCAG 2.1 AA compliance
- Keyboard navigation for all interactions
- Screen reader support (ARIA labels)
- Color contrast ratios > 4.5:1
- Focus indicators visible
- Alternative text for all images
- Captions for video content

---

## Loading & Error States

### Loading States
- Skeleton loaders for content areas
- Progress indicators for agent tasks
- Streaming text for AI generation

### Error States
- Inline validation messages
- Toast notifications for async errors
- Retry options for failed operations
- Clear error explanations

---

## Animation & Micro-interactions

- Page transitions: fade/slide (200ms)
- Button hover: scale 1.02 (150ms)
- Card hover: shadow elevation
- Agent status: pulse animation
- Progress: smooth fill animation
- Toast: slide in from top-right
